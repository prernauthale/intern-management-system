# Intern Management System

## 📌 Project Overview
This project is a full-stack Intern Management System built using:
- Spring Boot (Backend)
- Angular (Frontend)
- MySQL (Database)

## 🚀 Features
- Add Intern
- Generate Unique Intern ID
- Assign Batch
- Filter/Search Interns
- Batch Management

## 🛠️ Tech Stack
- Java (Spring Boot)
- Angular
- MySQL

## ▶️ How to Run

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
ng serve
